# Heal Hub

A community platform where users can share their healing journey through posts and images.

## Tech Stack
- React
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer

## Features
- User Signup
- User Login
- JWT Authentication
- Create Posts
- Upload Images
- Responsive UI

## Installation

Client
npm install
npm run dev

Server
npm install
npm run dev
